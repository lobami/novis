# Novis

![Novis Logo](https://via.placeholder.com/150x50/000000/FFFFFF?text=Novis)

**Novis** (from Latin *novus*, "new") is a highly experimental, modern programming language designed from the ground up for the era of AI and the Web. It features a clean, Python-inspired syntax, robust static type inference, native concurrency constructs, and first-class primitives for Artificial Intelligence (Tensors).

Written entirely in C++ (C++17) for blazing-fast compilation and execution, Novis aims to bridge the gap between elegant scripting languages and the raw performance needed for modern ML and system-level applications.

---

## ⚡ Features

- **Pythonic Syntax**: Clean, indentation-aware syntax. No braces, no semicolons required.
- **Blazing Fast Compilation**: Built as a header-only compiler infrastructure in C++17 that compiles in milliseconds.
- **Strong & Static Typing**: Enjoy the safety of static types with the ergonomics of dynamic typing through robust type inference (`x := 10` is inferred as `int`).
- **AI-First Primitives**: Native support for the `Tensor` type, preparing the ground for native automatic differentiation and MLIR/LLVM emission.
- **Built-in Financial Types**: Native `Decimal` and `Money` types for fintech precision.
- **Modern Concurrency**: (Planned) `spawn` and `await` primitives built directly into the language.
- **LLVM-Ready**: Includes a baseline IR Compiler (`--emit-ir`) to output intermediate representations for deeper compilation layers.

## 🚀 Quick Start

### 1. Installation

The easiest way to install Novis on macOS or Linux is via our universal install script:

```bash
curl -fsSL https://raw.githubusercontent.com/lobami/novis/main/install.sh | bash
```

*Note: You can also compile it from source. See the "Building from Source" section below.*

### 2. The REPL (Interactive Mode)

Simply type `novis` in your terminal to start the interactive Read-Eval-Print Loop:

```bash
$ novis
Novis 0.3.0 — type 'exit' or Ctrl-D to quit
>>> name := "World"
>>> print("Hello, " + name + "!")
Hello, World!
>>> 
```

### 3. Running Scripts

Create a file named `hello.novis`:

```python
# hello.novis
age := 25
money: Money = 100.50 USD
tensor_data: Tensor = [1.0, 2.5, -3.14]

if age > 18:
    print("Welcome!")
    print(tensor_data)
```

Run it using the Novis interpreter:

```bash
novis hello.novis
```

### 4. Emitting IR (Intermediate Representation)

To see the underlying PyraIR (Novis IR) generated by the compiler before native execution:

```bash
novis --emit-ir hello.novis -o out.pir
```

## 🛠 Building from Source

Novis is incredibly easy to build. You only need a modern C++ compiler (`clang++` or `g++`) supporting C++17.

```bash
# Clone the repository
git clone https://github.com/lobami/novis.git
cd novis

# Compile
make

# Run tests
make test
```

## 📖 Language Tour

### Variables & Inference
Novis uses `:=` for type-inferred assignment, and `:` for explicit typing.

```python
x := 42               # Inferred as int
y: double = 3.1415    # Explicitly typed
is_active := true     # boolean
```

### Functions

```python
fn calculate_area(width: double, height: double) -> double:
    return width * height

area := calculate_area(10.0, 5.5)
```

### Tensors (AI Primitive)

```python
weights: Tensor = [0.1, 0.2, 0.5, 0.9]
# Matrix operations and Autograd are planned for v0.5
```
